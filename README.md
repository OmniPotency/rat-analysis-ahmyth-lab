# RAT Analysis Lab — AhMyth (Educational)

**Course:** Intro to Cyber Security Lab (CY104L) — Air University
**Supervised by:** Ms. Ukasha Shahid Paracha

## Overview
A hands-on lab analyzing how Remote Access Trojans (RATs) operate, using the open-source Android RAT tool AhMyth in a fully isolated virtual environment (Kali Linux + Android emulation). The goal was to understand the attacker's perspective — how a RAT is deployed, how it gains persistence, and what data it can access — in order to build stronger defensive intuition.

## What's covered
- Setting up an isolated attack/victim lab using virtual machines
- Binding a RAT payload to a legitimate-looking APK
- Simulating full device compromise (files, call logs, camera, location) in a controlled, offline lab setting
- **A full defensive section**: how RATs typically spread, warning signs of infection, and incident response steps if one is suspected

## ⚠️ Responsible Use Notice
This project was conducted entirely within isolated virtual machines with no real devices, real user data, or real networks involved. AhMyth is an open-source tool intended for authorized security research and education only. This repository documents defensive learning, not an operational guide — do not use these techniques outside a controlled, authorized lab environment.
